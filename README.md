# SpriteSheetAnimation
<ol>
  <li>Цей проєкт демонструє просту систему анімації персонажа <mark>на основі спрайтшітів у p5.js</mark>.</li>
  <li>Формат спрайтів — прямокутні кадри розміром <mark>32x32 px</mark>, які масштабуються у грі за допомогою слайдера (від ×1 до ×5).</li>
  <li><mark>Анімації мають різну швидкість:</mark> Idle (6 fps), Walk (10–12 fps), Run (14–16 fps), а також окремий режим Jump (12 fps).</li>
  <li><mark>Керування здійснюється клавішами зі стрілками для руху</mark>, клавішею F або кнопкою для фліпу персонажа, а також кнопками 1, 2, 3 для перемикання між Idle, Walk і Run; пробіл активує стрибок.</li>
  <li><mark>Додатково передбачено hitbox</mark> для перевірки зіткнень, який відображається як червоне коло.</li>
  <li><mark>У майбутньому можна покращити проєкт, додавши плавні переходи між анімаціями</mark> (наприклад, автоматичне повернення з Jump у попередній стан), розширити набір кадрів для більш реалістичних рухів та інтегрувати систему зіткнень із об’єктами сцени.</li>
</ol>
